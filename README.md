# 🎯 Avito Sniper Bot (Smart Smartphone Parser)

Снайперский парсер Авито, созданный для поиска недооцененных флагманских смартфонов (Motorola, OnePlus, Google Pixel, Nothing Phone) по низу рынка. 

В отличие от обычных парсеров, этот бот **не просто собирает ссылки**, а работает как профессиональный перекуп: он читает текст, анализирует скрытые дефекты, проверяет рейтинг продавца и выставляет каждому объявлению оценку "Крутости" от 0 до 10 баллов.

## 🚀 Фишки (Features)
* **JS DOM Scanner:** Пробивает защиту Авито и вытягивает статус "Состояние" и "Экран" напрямую из интерфейса, игнорируя скрытые HTML-теги.
* **Scoring System (Умная оценка):** Начисляет баллы за идеальное состояние, наличие родной зарядки и выявляет "Джекпоты".
* **Anti-Scam Filter:** Автоматически отбраковывает заблокированные телефоны (MDM, Demo, iCloud), разбитые экраны и анализирует профиль продавца (рейтинг и дата регистрации).
* **Geo & Delivery:** Умный фильтр доставки. Исключает лоты "без доставки", если они находятся не в вашем городе.
* **Battery & Kit Extraction:** Достает из описания процент остатка АКБ и определяет полноту комплекта.

## 🛠 Установка

1. Склонируйте репозиторий:
```bash
git clone [https://github.com/ВашНик/avito-sniper-bot.git](https://github.com/ВашНик/avito-sniper-bot.git)
```
2. Установите необходимые библиотеки Python:
```bash
pip install playwright openpyxl
```
3. Установите встроенные браузеры Playwright:
```bash
playwright install
```
# Использование
Запустите главного бота для начала сбора (откроется окно браузера, при появлении капчи решите её вручную):
```bash
python avito_parser.py
```
После завершения работы в папке появится файл avito_results.xlsx.
Для быстрого просмотра ТОП-результатов прямо в терминале запустите читалку:
```bash
python check_results.py
```
