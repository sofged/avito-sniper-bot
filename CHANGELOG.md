## [Unreleased]

### 🚀 Added
* Добавлена лицензия AGPL-3.0 для обеспечения правового статуса проекта.
* Реализован механизм кеширования данных для повышения производительности и снижения нагрузки на целевые ресурсы.
* Расширен список поддерживаемых моделей OnePlus в модуле парсинга.
* Внедрена проверка условия «Только обмен» в описание объектов для уточнения критериев отбора.

### ⚙️ Changed
* Модернизирована логика первого этапа парсинга: исключена возможность перезаписи модели устройства при обработке пустых страниц.
* Оптимизированы алгоритмы фильтрации моделей: добавлена поддержка обработки слитного и раздельного написания наименований.
* Увеличены параметры задержек (timeouts) и максимального количества страниц для повышения стабильности при работе с динамическим контентом.
* Обновлено оформление документации README для улучшения навигации и актуальности технической информации.

### 🐛 Fixed
* Устранена ошибка в регулярных выражениях, отвечающих за корректное распознавание параметров джекпотов.

### 🛡️ Security
* Текущий уровень безопасности оценен как оптимальный (security_score: 100). Проведен аудит статического анализа, выявлены риски концентрации знаний и фрагментации модулей (orphan module status), требующие внимания при дальнейшем рефакторинге.
